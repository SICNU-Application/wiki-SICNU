### **面试情况**
#### **西安电子科技大学**
这是我最先联系成功并面试的学校，他们的推免通知在9月初发布，刚好跟我们学校的保研公示名单的时间比较接近，就很顺利的联系上了好几个老师。在与一个老师沟通后，确认了面试时间，是线上面试。面试流程依次是5分钟左右英语自我介绍、个人简历讲解、最后是对团队的了解以及实践项目中相关工作在他们团队方向中的应用。这种面试起来压力不是太大，看不见对面的情况，而且跟老师商量好后面试通过的概率特别大。

#### **电子科技大学**
这是我在西电之后联系成功并面试的学校，他们主要在优本计划（8月底左右）就招收的差不多了，我当时不太清楚自己是否能保研，处于一个没去管的状态，导致错过了这个很好的时间段，当时也只看了通信院，没有看其他学院，就这样浪费了优本计划这个机会。其次它还有一个开推免系统前几天的九推，当时也是成功联系上一个电院的老师，并且线下面试了，面试流程依次是5分钟左右英语自我介绍、两个英语问答、接下来就是五个专业课知识；关于专业课其实都还问的挺简单的，主要抽查的是电磁场的知识，原理啥的记一下就好了。但我在英语部分表现较差，被刷。

#### **重庆大学**
我在联系成电的同时也在联系重大，他们的推免通知也是比较晚的，并且面试时间在19号的样子（22号开系统填报），由于重大是弱com，需要先由学院统一面试后再去选老师，感觉这一点也还挺不错的，能避免许多的已读不回。我和一个老师联系好后填报了系统，重大也海放了很多面试通知，面试需要提前交500元并且19号面试，但必须要18号早上进行线下报道，这个感觉就是排除掉一些拿重大当备选的学生，而成电的面试是在17号，时间上我有一点赶就没去。

#### **西南交通大学**
他们的面试时间应该是最晚的，我选择了三个评价最好的老师，成功联系上了，也收到一个老师团队的面试通知。21号面试，20号晚上才创建对应群，整个面试过程是比较水的，有的老师团队的QQ群能全要群中的30多个人，我选那个老师他带的团队有好几个群，每个群都差不多60个人左右，时间也是非常的紧迫，一个人只给3-5min介绍自己，我当时介绍完后那个老师问了我几个简历中提到的比赛，但优势不大，也就面试被刷掉了。

### **信息建议**
尽早找好自己想去的一些学校，并列好清单，哪怕不能保研也方便后面考研择校。一定要尝试往好学校冲刺的机会，尽早投递邮件的话捡漏的机会也就越大。中上985也要试试，不要把目光只留在电科、重大、川大、西南交大这些川渝地区的学校，把握好时间。先就是海投邮件，一定要做好很多老师不看邮箱或者已读不回的准备，在成功与某个老师取得联系后再去查看老师的团队信息，搜一下老师的风评怎么样（研控上可以搜索），的如果老师风评不好，压榨学生啥的，尽早去投其他老师。

### **保研前准备**
1. 明确保研分数组成：学业成绩的80% + 量化成绩的20% + 加分项
2. 保研的基本要求：至少两次学业奖学金 + 一次荣誉称号 + 英语四级通过
3. 认真对待每门专业课，争取取得一个好成绩。看的是前六学期的学业成绩，不要松懈，也不要觉得自己一开始成绩不怎么好就放弃，我也有认识好几个人都是大二开始成绩才一下好起来的，也能拿到不错的名次
4. 千万不要挂科，挂科了的话几乎就与保研无缘了，我们这一届通信专业的专业第一和另一个（忘记是专业第五还是第六）就是因为在大三上挂了一科直接就没有了保研资格，不到最后一刻千万不要松懈
5. 研究生老师更看重的是科研能力，或者你有一个很好的专业排名，在rank1或者rank2，所以本科阶段最好能加入一个实验室跟着学习与比赛，多与老师沟通，跟着老师参与一些项目与比赛，提高自己的科研水平
6. 对于保研材料中的加分项，感觉比较能实现的就是打比赛、申请专利以及发论文，其实大一就可以去混迹实验室，跟着学长学姐参加比赛，哪怕没有奖，也可以先熟悉比赛流程、知道比赛中的应急措施以及临场能力；专利和论文的难度还是比较大的，但如果积极联系老师、积极参与老师手中的项目还是能够有产出的。

### **保研面试准备**
1. 先找到想去的学校，不要好高骛远，也不要自我贬低。根据自身情况以及往届学生的保研情况划分自己去的学校的难易程度，每个难易程度建议都选2个左右的学校。
2. 做备好自己的简历、套话，对想去的学校进行海投（跟自己搭边的专业）；要着重体验自己的排名、科研经历、学术论文以及竞赛经历。一定要查清楚去的学校的对应学院需要提交哪些文件，并且弄清楚这个学院是强com还是弱com。
3. 大致复习一下专业课知识，在心中有一个底，对于一些基本概念能有自己的理解。对于面试所涉及的专业课建议整体过一遍，对每一个知识点加深印象。
4. 面试的时候保持冷静、从容对待，不要不懂装懂，真诚回答提问的问题。

## 寄语
心有所向，步履不停.你所走过的每一步都会为之后铺路，保研是一场长跑，需要用三年的踏实完成水到渠成。踏踏实实的学好每一门课程，相信积累的力量，哪怕是起初的为了奖学金、做好自己的本职工作，完成学业任务，都在不知不觉中塑造了你的履历与能力。不要轻视任何一次努力，它们终将在某个关键节点，成为你的底气和光芒。保研是行动与信息的结合，尽早规划，主动收集信息。不要自我设限，也不要好高骛远，科研之路漫长，找一个好团队、好导师，远比一个听起来响亮的头衔更重要。

保研是一场持久战，有焦虑、有等待、也有柳暗花明的喜悦。或许你会遇到已读不回的石沉大海，会遇到紧张导致的失误，也会在多个选择间徘徊——这一切都是常态，你并不孤单。你的价值也绝不局限于一张录取通知。在这段旅程中，你锤炼出的抗压能力、信息搜集能力、临场应变能力，以及对自己更清晰的认知，都是终身受用的财富。
